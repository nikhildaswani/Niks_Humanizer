# Humanizer Skill

Humanizer is a writing-editing skill that removes common AI writing patterns and
rewrites text so it reads more naturally while preserving the author's meaning,
facts, and level of confidence.

## What It Does

- Detects common AI writing patterns such as significance inflation,
  promotional tone, vague attributions, filler phrases, rule-of-three padding,
  synonym cycling, excessive hedging, and chatbot artifacts.
- Adapts edits to the writing context: professional, creative, technical,
  academic, or casual.
- Preserves the original voice instead of imposing a new one.
- Avoids invented facts, citations, dates, statistics, feelings, or personal
  experience.
- Uses an adversarial audit pass for substantial rewrites.

## Usage

Use this skill when you want to:

- clean up AI-generated or AI-assisted writing
- make text sound more natural without changing its meaning
- remove generic chatbot tone, puffery, and formulaic structure
- preserve the original author's voice while improving readability

This skill is designed to edit text, not to fabricate detail. If the source is
vague, the rewrite should stay vague or become more carefully qualified.
